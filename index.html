Gulpsetup

$ gulp serve
  • Structured Gulpfile using partials
  • Compile Sass using gulp-sass (Libsass)
  • Concatinate and minify Javascript using Uglify
  • Synchronised browser testing using BrowserSync and xip.io
  • Autoprefixer
  • Some helpers: plumber, size, csslint, jshint, size report, image optimizations etc. See readme.
